Exercise 5: Lamp

Problem: Create a model of the a lamp image using NURBS tools

We were given a single reference image of a lamp to model. I began the process by creating a bezier curve to draw out the cross section of the lamp. I used the revolve tool to turn this into a NURBS surface, and then went back to the original curve to refine the shape a bit. I repeated this same process for the lamp shade. For the base platform, I created multiplie NURBS squares of different sizes (each representing cross sections of the platform at points where the shape changes), and used the loft functions to build a surface.

Exercise 4: Table

Problem: Create a detailed model of a work desk

We were given a single reference image of a work table to model. I began the process with box modeling, and worked with the basic set of modeling tools like the extrude tool and the multi cut tool. I had to restart the legs several times to get it to look right. The biggest difficulty I had was to try and minimize unnecessary support edge loops. This involved working and reworking the model's topology.

Exercise 3: Lemon Squeezer

Problem: Create a detailed model of a lemon squeezer

I started out by creating a cylinder to model the squeezer. I had start and restart a few times to figure out how to get the propper shape. I ended up extruding faces to create the indentation of the squeezer, and added edge loops to pull the sharp edges out. Creating the base was a lot easier, and just involved adding edge loops and moving vertices. I then went back and cleaned up the model to make sure I didn't have any ngons or triangles.

Exercise 2: Solar System

Problem: Create a solar system with a sun, planets and moons. Animated the planets and moons to rotate and orbit as required

For this assignment, I decided to stick to spheres (and a torus for Saturn's ring) and use textures to distinguish between the different planets and moons. I found texture maps for these objects from various sources online, and applied them as the color map on lambert shaders. I decided to avoid using manual keyframes for the animation. Instead, I felt that I could save a lot of time by using expressions to control the rotation. I had all the planets and moons grouped in a way that made it easy to add expressions appropriately. All the objects rotate and orbit at unique speeds.
