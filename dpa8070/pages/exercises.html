Exercise 9: Shading and Lighting

Problem: Create two spheres: orange and jupiter. Create shaders and lighting to match the pictures as close as possible.

For both of the objects, I started out by applying an aiStandardShader, and creating 3 point lighting for the scene using a Maya directional light for the key light, and an Arnold area light for the fill and rim lights. For shading the orange, I set the base color and added Brownian noise as a bump map. For Jupiter, I started by creating a ramp node and adding various colors to represent different parts of the planet. I tried to add the vortex by using Maya’s leather texture and combining both in a layer node. I used this as the base color for Jupiter. I then used some wispy noise as a displacement map.

Exercise 7: NURBS Shoe

Problem: Finish modeling the given shoe using NURBS Tools

To model this shoe, I began by lofting Bezier curves to form the base of the shoe. I beveled the edge to of the top to make it more aesthetically pleasing. I used the extrude tool to form the straps, and the resolve tool to create the heel and the back of the shoe. I finally split my curves into halves and lofted them together to cap the open surfaces.

Exercise 6: Watch Tutorials

Problem: Watch 10 lessons of a tutorial series on a specific process in Maya and briefly describe what you learned

For this exercise, I found a course titled “ Learn Maya - Character Head Modeling for Beginners ” on Udemy taught by 3D Character Instructor Nalini Kanta Jena. The 1st two videos covered prepping and setting up reference images on image planes in Maya. The 3rd video went over the anatomy of eyeballs and demonstrated the approach to model them using a combination of 2 different deformed spheres. The 4th video covered the technique to create eyelids by using the Create Polygon Tool and extruding edges. The 5th video went over blocking the basic lip shapes using the same technique as eyelids. The 6th video illustrated modeling the cheek and connecting the mouth to the eyes. The 7th video discussed blocking out the forehead and overall shape of the skull. One useful technique demonstrated was enabling the Insert with Edge Flow option in the Insert Edge Loop tool. The 8th went over creating the neck by extruding and rotating edges to create a bust of sorts. The 9th video covered the anatomy and constructing of nostrils, completing the rough model. The 10th video started the next unit which deals with detailing. This video demonstrated how to detail the lips by using appropriately positioned edge loops to both add details and support the existing edges to harden them as required. My biggest take away from this series was the importance of understanding anatomy before beginning a model.

Exercise 5: Lamp

Problem: Create a model of the a lamp image using NURBS tools

We were given a single reference image of a lamp to model. I began the process by creating a bezier curve to draw out the cross section of the lamp. I used the revolve tool to turn this into a NURBS surface, and then went back to the original curve to refine the shape a bit. I repeated this same process for the lamp shade. For the base platform, I created multiplie NURBS squares of different sizes (each representing cross sections of the platform at points where the shape changes), and used the loft functions to build a surface.

Exercise 4: Table

Problem: Create a detailed model of a work desk

We were given a single reference image of a work table to model. I began the process with box modeling, and worked with the basic set of modeling tools like the extrude tool and the multi cut tool. I had to restart the legs several times to get it to look right. The biggest difficulty I had was to try and minimize unnecessary support edge loops. Exercise 3: Lemon Squeezer

Problem: Create a detailed model of a lemon squeezer

I started out by creating a cylinder to model the squeezer. I had start and restart a few times to figure out how to get the propper shape. I ended up extruding faces to create the indentation of the squeezer, and added edge loops to pull the sharp edges out. Creating the base was a lot easier, and just involved adding edge loops and moving vertices. I then went back and cleaned up the model to make sure I didn't have any ngons or triangles.

Exercise 2: Solar System

Problem: Create a solar system with a sun, planets and moons. Animated the planets and moons to rotate and orbit as required

For this assignment, I decided to stick to spheres (and a torus for Saturn's ring) and use textures to distinguish between the different planets and moons. I found texture maps for these objects from various sources online, and applied them as the color map on lambert shaders. I decided to avoid using manual keyframes for the animation. Instead, I felt that I could save a lot of time by using expressions to control the rotation. I had all the planets and moons grouped in a way that made it easy to add expressions appropriately. All the objects rotate and orbit at unique speeds.
